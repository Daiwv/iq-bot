#iq-bot(Python)

###Required:
1. selenium
2. pywin32

###Usage:
./iq.py
..* -u <'Email пользователя>
..* -p <Пароль пользователя>
..* -m <Выбор режима работы [demo,real,test]>
..* -o <Выбор опциона [turbo,bin]>
..* -l <Выбор языка MT alert окна [eng,rus]>
..* -a <Выбор актива [EUR/USD, BITCOIN]>

####Example:
./iq.py -u user_name -p password
